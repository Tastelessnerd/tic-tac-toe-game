# Tic Tac Toe Game
A simple browser-based Tic Tac Toe game built using **HTML, CSS, and JavaScript**.
## Features
- Two-player game (Player X and Player O)
- Winner detection
- Draw detection
- Restart game button
- Clean and simple user interface
## Technologies Used
- HTML
- CSS
- JavaScript
## How to Run
1. Clone the repository
2. Open `index.html` in your browser
## Live Demo
https://tastelessnerd.github.io/tic-tac-toe-game/
## Project Purpose
This project was built to practice **JavaScript DOM manipulation and basic game logic**.
